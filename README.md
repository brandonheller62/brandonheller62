### 🚀 Featured projects

#### [Ransom Tutor](https://github.com/brandonheller62/ransom-tutor) · [Live demo →](https://ransomtutor.vercel.app)
An AI Socratic tutor for two real high-school courses — Advanced Physics (Mechanics) and
Applied Data Science. Instead of giving away answers, it guides students with hints and
questions, and every explanation is **grounded in the actual course syllabi** using
retrieval-augmented generation (RAG), so it stays accurate to what's being taught.

It includes a guided chat tutor, auto-generated practice quizzes (multiple-choice and
free-response with grading), progress tracking, rendered math (KaTeX), and image uploads
so students can submit handwritten work for feedback. It runs on a dual-provider model
backend (Anthropic Claude, with an OpenAI fallback) and is deployed and verified in
production.

`Next.js · TypeScript · RAG · Anthropic Claude · OpenAI · Supabase (pgvector) · Voyage embeddings · Tailwind`

#### [Calculus Coach](https://github.com/brandonheller62/calculus-coach)
An AI tutor focused on AP Calculus AB — step-by-step problem solving, quizzes, and
interactive practice designed to build genuine understanding of the concepts the exam
tests, not just memorized procedures.

`Next.js · TypeScript · LLM-powered tutoring`
